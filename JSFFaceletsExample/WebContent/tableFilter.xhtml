Table Menu - alternatywa przeszukiwania taksonomii
  • Mozliwosc filtrowania taksonomii po nazwach
  • Bedzie implementacja dynamicznych taksonomii - uaktualnianie ilosci mozliwych wynikow na biezaco
  • Jest mozliwosc okreslenia priorytetu dla taksonomii

Okreslonych jest 5 poziomow priorytetow przy wyborze elementow taksonomii:
  • Najwyzszy [1] - wybrany element musi byc uwzgledniony
  • Wysoki [2] - wybrany element jest preferowany
  • Normalny [3] - wybrany element ma normalny priorytet
  • Pomijany [4] - wybrany element nie jest preferowany na liscie wynikow
  • Zakazany [5] - wybrany element nie moze byc na liscie wynikow
